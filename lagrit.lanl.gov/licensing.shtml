LaGriT Registration

Supported platforms are Mac OSX (Intel only) and Linux. Windows distribution will be available soon. An exacutible is distributed that does not require compiling or building.

If you obtain the source code for your own user modules, it is suggested that you compile it with gfortran and gcc Versions 4.5 or greater. If you use another compiler and have problems, we will not be able to help.

LaGriT now supports ExodusII file format output (which sits on top of netCDF). If you want to compile your own code, you have to build the ExodusII and netCDF libraries yourself. We do not distribute those packages. Contact Greg Sjaardema at Sandia National Laboratories (gdsjaar @ sandia.gov):

http://sourceforge.net/projects/exodusii/files/

  • * ExodusII 5.22b
  • * HDF5 version 1.8.6
  • * netcdf-4.1.3
