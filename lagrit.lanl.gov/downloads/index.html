Feel free to send email if you have questions. Let us know if you have any problems or would like assistance getting started.

General documentation for LaGriT can be found at the URL: lagrit.lanl.gov

Documentation for each LaGriT module can be found at the URL: lagrit.lanl.gov/manual.shtml

Some examples and meshing projects can be found in: meshing.lanl.gov/proj

To run LaGriT just type: lagrit_YOUR_OS_o and enter commands from the command line.
For example at the first command prompt you can type: infile demo.lgi
To run in background from a LaGriT control file type:
lagrit_YOUR_OS_o < demo.lgi
